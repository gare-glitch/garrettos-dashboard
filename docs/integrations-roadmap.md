# Integrations roadmap

## Garmin

Start with manual CSV uploads, then add server-side FIT/TCX parsing and scheduled imports.

## Obsidian

Add vault upload or sync jobs, then chunk notes into memory records for search and AI Mentor context.

## OpenClaw

Connect server-side status polling first. Require explicit approvals before any privileged agent action.

## VPS monitoring

Add a signed server-side metrics collector for Ollama, LiteLLM, Qdrant, Valkey, Docker, CPU, RAM, and disk.
